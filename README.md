# Redoc UI Plug

The collection of Plug modules to render Redoc UI from OpenAPI spec. Inspired 
by [open_api_spex](https://github.com/open-api-spex/open_api_spex).

## Installation

Adding `redoc_ui_plug` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:redoc_ui_plug, "~> 0.1.0"}
  ]
end
```
